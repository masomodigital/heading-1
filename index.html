Lesson: Creating a Heading with <h1>

Introduction

Welcome to your first HTML coding lesson! Today, we're going to learn how to create a heading in HTML. A heading is like a title that appears at the top of a page. It helps people know what the page or section is about. In HTML, we use special tags to create different parts of a webpage, and to make a big, bold heading, we use the <h1> tag.

Explanation

The <h1> tag is used to make the biggest and most important heading on a webpage. The text inside this tag will appear larger and bolder than regular text, so it stands out. Think of it like the title of a book; it should be easy to see and read.

Here’s how you write an <h1> tag:

<h1>This is a Heading</h1>

In the example above, "This is a Heading" is the text that will appear big and bold on the page. You can put any text you like between the <h1> and </h1> tags.

Example

Let’s see an example in code:

<!DOCTYPE html>
<html>
<head>
    <title>My First Heading</title>
</head>
<body>
    <h1>Welcome to My Webpage</h1>
</body>
</html>

When you run this code, you'll see "Welcome to My Webpage" appear as a large, bold heading at the top of the page.
